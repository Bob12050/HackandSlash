# こもれびギルド物語

TypeScriptとPhaser 4で作る、スマートフォン向けの可愛い縦型自動戦闘RPGです。

[GitHub Pagesで遊ぶ](https://bob12050.github.io/HackandSlash/)

## ゲームの流れ

1. ギルドから「冒険に出る」を選び、行き先を決める
2. 冒険先での自動戦闘を見守る
3. そよかぜ草原で10体倒し、「おおきな王冠スライム」に挑む
4. ボスを倒して「こもれび森」を解放し、より上質な装備を集める
5. ギルドへ帰還し、装備の付け替えや鍛冶屋での強化を行う
6. クエストを達成しながらSランク冒険者を目指す

歩行や複雑な操作はなく、タップだけで遊べます。冒険の記録はブラウザへ自動保存されます。

## 現在のプレイ要素

- ギルドと冒険画面を切り替える縦型モバイルUI
- 敵との自動戦闘、ダメージ演出、敗北時の自動帰還
- 冒険先の選択、エリアごとの討伐進行、ボス戦
- 草原ボスの初回討伐で「こもれび森」と限定装備を解放
- こもれび森では敵が強くなる代わりに、装備のドロップ率と品質が上昇
- 経験値、レベルアップ、Gold、全101種類の装備ドロップ
- 草原・森ごとに異なる装備プールと、ゲーム内の装備図鑑
- 装備の比較・付け替え・売却
- 鍛冶屋でGoldを使い、所持装備を最大+10まで強化（強化段階に応じて費用が上昇）
- 討伐クエストと報酬受け取り
- ローカルストレージによるオートセーブ（討伐数・エリア解放も保存）

## ローカルで起動

```bash
npm install
npm run dev
```

表示されたURLをブラウザで開いてください。

## 検証

```bash
npm run test
npm run typecheck
npm run build
npm run validate
```

## 技術構成

- TypeScript 7
- Phaser 4.2.1
- Vite 8
- Vitest 4
- Font Awesome Free

`main` ブランチへpushすると、GitHub Actionsが検証とビルドを行い、成功後にGitHub Pagesへ自動公開します。
